# Codex project import notes

This bundle is prepared for OpenAI Codex.

## Included Codex files
- `.codex/config.toml` for project-scoped Codex settings
- `AGENTS.md` for repository instructions and working agreements

## How to use
1. Unzip this archive.
2. Open this project folder as your project in Codex.
3. Trust the project so Codex can load `.codex/config.toml`.
4. Start Codex from the repository root.
