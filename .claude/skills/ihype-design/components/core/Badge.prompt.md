# Badge

Small inline label for roles, states, or categories. Use for role tags (Fan/Artist/Venue/Promoter), status chips, or track metadata.

```jsx
<Badge color="#ff5029">Artist</Badge>
<Badge color="#b983ff">Fan</Badge>
<Badge color="#22e5d4" variant="outline">Venue</Badge>
```

## Notable variants
- `color` — any hex; controls background tint (10% opacity) and text color
- `variant="filled"` — default; 10% opacity fill
- `variant="outline"` — transparent bg, 30% opacity border
