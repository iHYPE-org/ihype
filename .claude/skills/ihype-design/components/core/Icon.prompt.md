# Icon

Renders any Lucide icon by name. Lucide must be loaded in the page head first.

```html
<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.min.js"></script>
```

```jsx
<Icon name="music" size={20} />
<Icon name="heart" size={16} color="#ff5029" />
<Icon name="calendar" size={14} color="#22e5d4" strokeWidth={1.4} />
```

## Common iHYPE icons
- `music`, `radio`, `headphones` — audio
- `heart`, `zap`, `flame` — engagement / HYPE
- `calendar`, `map-pin`, `ticket` — shows
- `user`, `users`, `shield` — identity
- `bar-chart-2`, `trending-up` — analytics
- `settings`, `lock`, `eye` — privacy/ops
