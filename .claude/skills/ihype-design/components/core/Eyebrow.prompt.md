# Eyebrow

All-caps JetBrains Mono label used for section headers, metadata chips, and contextual timestamps. The "technical readout" voice of iHYPE.

```jsx
<Eyebrow>TONIGHT · 9:00 PM · DOORS OPEN</Eyebrow>
<Eyebrow color="#22e5d4">AT EMPTY BOTTLE</Eyebrow>
<Eyebrow color="#ff5029">● HYPE FIRES AT 3:38</Eyebrow>
```

## Notable variants
- `color` — defaults to `--ink-3` (#5a5048); pass role color for emphasis
- Dot separator `·` — use as the standard between metadata tokens
- Bullet `●` — use to prefix live/active state (HYPE fires)
